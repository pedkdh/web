# 테스트환경 : 아파치 웹서버, PHP 5.x 이상, 아치리눅스

[설치방법]

기본적으로 웹서버와 PHP 가 설치되어 있어야 합니다.
pogo_index.zip 파일의 압축을 풀고, pogo_index 폴더의 내용을 웹서버 루트 디렉토리에 업로드하면 됩니다.

설치 예)

웹서버 루트 
	⊢- index.php (PC용 인덱스)
    ⊢- header.php (PC용 헤더파일)
    ⊢- m_index.php (모바일용 인덱스)
    ⊢- info.php (PHP 버전확인용)
    ⊢- style.css (PC용 CSS)
    ⊢- images (모바일용 이미지디렉토리)
    ⊢- gallery (Juicebox용 인덱스 디렉토리)
    	⊢- index.php
    	⊢- m_index.php
    	⊢- main.css
    	⊢- read_album.php
    	⊢- pix (이미지 디렉토리, 폴더명을 변경하지 마세요)
    ⊢- time (PC용 인덱스 파일에서 사용, 시계함수)


[Juicebox용 인덱스 파일 사용하기]

gallery 디렉토리내에 있는 파일들은 웹갤러리인 Juicebox용 인덱스 파일입니다.
Juicebox는 SQL을 사용하지 않기 때문에 가볍고, PC에서 앨범을 제작하여 업로드하기 때문에 썸네일 이미지를 만들기 위해 서버 측에 부하를 주지 않습니다.
(특히나 메모리가 작은 시리즈4에 적합)

Juicebox 인덱스 파일은 Juicebox 에서 만든 XML파일을 읽어들여서 동작합니다.
현재 디렉토리(.), 상위 디렉토리(..)와 인덱스 파일에서 사용하는 디렉토리(pix)는 읽지 않도록 되어있습니다만, 혹 현재 갤러리 디렉토리가 상단에 나타나거나 한다면
for($i=0;...) 이 부분을 $i=1 로 바꾸어서 해보시기 바랍니다.

Juicebox로 만든 앨범이 업로드되어 있는 디렉토리에 gallery 디렉토리 내의 파일과 폴더를 업로드하면 됩니다.
index.php 파일 상단의 경로 부분을 본인의 경로에 맞게 수정하시기 바랍니다.

header("Location: http://$_SERVER[SERVER_NAME]/photos/m_index.php"); (photos 부분을 본인 경로에 맞게 수정)
